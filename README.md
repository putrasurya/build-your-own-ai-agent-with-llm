# Build Your Own AI Agent with LLM

This repository contains code examples related to the Medium blog post: [Build Your Own AI Agent with LLM](https://medium.com/@putrasurya/build-your-own-ai-agent-with-llm-model-30f2bca0e17c).

## Overview
The blog post demonstrates how to create your own AI agent using Large Language Models (LLMs). It is divided into three sections, each with corresponding code examples in this repository:

1. **A Simple Chat**: Demonstrates the basics of interacting with an LLM to create a simple chatbot.
2. **A Tool Call in LLM**: Demonstrates how LLM can call tools to enhance its capabilities.
3. **A Technical Support Agent**: Implements a simple AI agent that acts as a technical support assistant, showcasing a more practical use case.

## Files
- `01_simple_chat.py`: Code for the simple chat example.
- `02_tool_calling_example.py`: Code for the tool-calling example.
- `03_technical_support_agent.py`: Code for the technical support agent example.

## How to Use
1. Clone this repository.
2. Install the required dependencies (if any) using `pip install -r requirements.txt`.
3. Run the Python scripts to explore each example.
